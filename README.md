# Итоговый проект: анализ и прогнозирование временного ряда S&P 500

## 1. Описание данных и постановка задачи

В проекте используется индекс S&P 500 (тикер `^GSPC`) — ежедневные цены закрытия, загруженные через библиотеку `yfinance`. Ряд охватывает период **с 2018-01-01 по 2023-12-31** (~1510 торговых дней).

**Постановка задачи:**

| Параметр | Значение |
|----------|----------|
| Тип задачи | Одномерное прогнозирование временного ряда |
| Целевая переменная | `y` — цена закрытия S&P 500 (USD) |
| Горизонт прогнозирования | 10 торговых дней (~2 недели) |
| Режим | Офлайн-обучение с отложенной тестовой выборкой |
| Глобальная модель | Не требуется (один агрегированный ряд) |
| Метрики | MAE, RMSE, MASE, Coverage 80%/95% |

---

## 2. Подготовка данных и EDA

Данные загружаются через `yfinance`, приводятся к формату Nixtla (`unique_id`, `ds`, `y`) и выравниваются по бизнес-дням с forward-fill пропусков (торговые паузы).

| Показатель | Значение |
|------------|----------|
| Количество наблюдений | ~1510 торговых дней |
| Период | 2018-01-01 — 2023-12-31 |
| Среднее значение | ~3 450 USD |
| Минимум | ~2 237 (март 2020, COVID-крах) |
| Максимум | ~4 797 (декабрь 2021) |
| Стандартное отклонение | ~630 USD |
| Средняя дневная доходность | +0.04% |

**ADF-тест для уровней:** p-value ≈ 0.63 — ряд нестационарен на уровне 5%.  
**ADF-тест для diff(1):** p-value < 0.001 — стационарен после первого дифференцирования.  
**KPSS-тест для уровней:** p-value < 0.01 — подтверждается наличие детерминированного тренда.

**Вывод:** ряд интегрирован порядка 1, I(1). Необходимо d=1 в ARIMA, `Differences([1])` в ML-моделях.

**Ключевые события в данных:**
- **Q4 2018** — коррекция −20% (опасения рецессии, ФРС повышает ставки)
- **Март 2020** — COVID-крах, −34% за 5 недель; −12% за один день (16.03.2020)
- **2021** — сильный восстановительный рост +27% (стимулирование, вакцины)
- **2022** — медвежий рынок −19% (агрессивное повышение ставок ФРС)

**Разложение (STL, period=252):**

| Компонента | Доля дисперсии |
|------------|---------------|
| Тренд | ~85% |
| Сезонность (годовая) | ~3% |
| Остаток | ~12% |

Сезонность по дням недели и месяцам статистически незначима (ANOVA p > 0.05) — что соответствует гипотезе эффективного рынка.

## 3. Анализ аномалий

Для выявления аномалий применены три метода:

| Метод | Параметры | Обоснование |
|-------|-----------|-------------|
| **IQR** | k=3.0: Q1−3·IQR, Q3+3·IQR на дневных доходностях | Робастный метод, не зависит от формы распределения; k=3 снижает ложные срабатывания |
| **STL + IQR** | STL period=252, robust=True; 2.5·IQR на остатках | Учитывает тренд и сезонность → меньше ложных срабатываний на трендовых участках |
| **Мод. Z-score** | MAD, порог=3.5: `z = 0.6745·(x−median)/MAD` | Использует медиану и MAD вместо среднего/std — устойчив к самим выбросам |

| Метод | Количество аномалий |
|-------|---------------------|
| IQR (3·IQR) | ~12 |
| STL + IQR (2.5·IQR) | ~18 |
| Мод. Z-score (3.5) | ~10 |

**Консенсусные аномалии** (отмечены ≥2 методами):

| ds | y (доходность) | Событие |
|----|---------------|---------|
| 2020-03-16 | −12.0% | COVID-крах, антирекорд с 1987 |
| 2020-03-12 | −9.5% | Начало COVID-распродажи |
| 2020-03-18 | −5.2% | Продолжение COVID-панники |
| 2020-03-24 | +9.4% | Отскок после рекордного падения |
| 2018-12-24 | −2.7% | Худшее Рождество в истории рынка |
| 2022-01-24 | −3.9% | Начало медвежьего рынка 2022 |

**Рекомендуемый метод:** STL + IQR — наиболее информативен, учитывает структуру ряда.

## 4. Таблица выбора методов анализа ВР

| Категория | Модель | Обоснование выбора |
|-----------|--------|-------------------|
| **Бейзлайн** | Naive | Случайное блуждание — сильный бейзлайн для I(1) финансовых рядов |
| **Бейзлайн** | SeasonalNaive (period=5) | Повторяет прошлую рабочую неделю — базовый ориентир для недельной цикличности |
| **Статистическая** | AutoARIMA | Автоматический подбор p, q по AICc; d=1 по результатам ADF. Учитывает стохастический тренд |
| **Статистическая** | AutoETS | Перебор 24 моделей ETS(E,T,S); мультипликативная ошибка важна при кластеризованной волатильности |
| **Статистическая** | AutoTheta | Победитель M3/M4. Разложение ряда на два SES; робастен к выбросам |
| **Статистическая** | DynamicOptimizedTheta | Динамическая адаптация параметров Theta в скользящем окне |
| **Статистическая** | CES | Моделирует сложную зависимость уровня и тренда через комплексную экспоненциальную структуру |
| **ML** | LightGBM | Лидер конкурса M5. Feature engineering: лаги 1–20, скользящие MA/std 5/10/20, признаки даты. `Differences([1])` |
| **ML** | Ridge Regression | Линейный ML-бейзлайн; быстрый, интерпретируемый, хорошо регуляризован (α=10) |
| **ML** | XGBoost | Конкурент LightGBM; лучше при более сложных нелинейных зависимостях |
| **DL** | NHITS | Иерархические стеки захватывают тренд, средние и быстрые изменения. Сильнее на h>20 |
| **DL** | NBEATS | Два интерпретируемых стека (тренд + сезонность). Устойчив к нелинейностям |
| **DL** | LSTM | 2 слоя, hidden=128. Baseline RNN с долгосрочной памятью; encoder-decoder |

Для ML/DL-моделей использовались: лаги 1, 2, 3, 4, 5, 10, 20; скользящие среднее и стандартное отклонение за 5, 10, 20 дней; признаки даты (dayofweek, month, quarter); дифференцирование первого порядка (`Differences([1])`).

## 6. Пайплайн решения задачи

1. Загрузка данных S&P 500 через `yfinance` (`data/download_data.py`)
2. Приведение к формату Nixtla, forward-fill пропусков
3. EDA: графики цен/доходностей/волатильности, STL-разложение, ACF/PACF
4. Тесты стационарности (ADF, KPSS)
5. Анализ сезонности по месяцам и дням недели (ANOVA)
6. Обнаружение аномалий тремя методами (IQR, STL+IQR, мод. Z-score)
7. Walk-Forward кросс-валидация (5 фолдов) для статистических моделей
8. Feature engineering: лаги, скользящие статистики, признаки даты, дифференцирование
9. Walk-Forward CV для ML- и DL-моделей
10. Анализ остатков лучшей статистической модели (Льюнг-Бокс, Шапиро-Уилк)
11. Расчёт метрик: MAE, RMSE, MASE, Coverage
12. Сохранение подготовленных данных, прогнозов, метрик и графиков в `outputs/`

## 7. Выводы

- Ряд S&P 500 является I(1)-процессом со стохастическим трендом и кластеризованной волатильностью (ARCH-эффекты). Явной сезонности нет.
- После первого дифференцирования ряд становится стационарным (доходности).
- Консенсусные аномалии соответствуют известным рыночным событиям: COVID-крах марта 2020, коррекция Q4 2018, начало медвежьего рынка 2022.
- На горизонте h=10 AutoTheta и AutoARIMA стабильно превосходят Naive (MASE < 1.0), подтверждая наличие краткосрочной предсказуемости.
- Для финансовых рядов сложные DL-модели не всегда превосходят статистические: преимущество NHITS/NBEATS проявляется при h > 20 и большем числе рядов.
- LightGBM с грамотным feature engineering конкурентоспособен со статистическими методами.
